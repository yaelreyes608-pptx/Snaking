# Snaking (Still in development)

A "Snake" game application featuring Artificial Intelligence integration, competitive game modes, and a user management system.

## Features
- **Game Modes:** Player vs. Player, Player vs. AI, and AI vs. AI.
- **Access Control:** User authentication via Login system.
- **Technologies:**
    - **Frontend:** HTML5 Canvas / JavaScript.
    - **Backend:** Node.js.
    - **Database:** MongoDB.
    - **Deployment:** Heroku.

## Installation Requirements
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure environment variables (`MONGODB_URI`, `PORT`).
4. Start the application with `npm start`.
