# Snaking

## Features
- **Game Modes:** Single player Player vs. Player, Player vs. AI, and a scoreboard
- **Access Control:** User authentication via Login system.
- **Technologies:**
    - **Frontend:** HTML5 Canvas, CSS3, JavaScript.
    - **Backend:** Node.js, Express.js.
    - **DataBase:** MongoDB Atlas, Mongoose.
    - **Security:** Passport.js, Bcrypt.js, Express-Session.
    - **Deployment:** Heroku: https://snaking-f27c34bb4a45.herokuapp.com/

## Local setup
1. Clone or download the repository.
2. Open the folder in the console commands.
3. Execute the command "npm install" to download the dependencies.
4. Create an .env file with you access credentials to your database.
5. Run the command "node server.js" on the console with the folder open.
6. Search localhost:3000
